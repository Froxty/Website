Color Match

Color Match is a competitive color-matching mini-game built in Roblox. Players are given a target RGB color and adjust Red, Green, and Blue sliders (0–255) to match it as closely as possible before the timer runs out.

Project Overview

Color Match was developed in Roblox Studio as a systems-focused mini-game. The goal was to build a complete loop that combines UI interaction, server-side scoring, and persistent player stats, while collaborating with a dedicated UI designer.

Core Gameplay

Each round, the server generates a random RGB color. Players see that target color and use three sliders (R, G, B) to match it, with values ranging from 0–255. A countdown timer limits how long they have to adjust their guess.

When the round ends or a player hits Submit, the server calculates how close their guess is and displays an accuracy percentage on a summary screen.

Data & Progression

The game uses Roblox DataStore to persist stats between sessions. For solo play, it currently tracks:

  • Total games played
  • Average accuracy percentage
  • Average time taken to submit

These stats are surfaced in a dedicated Stats menu so players can see their improvement over time.

Achievements

Color Match includes an achievement/badge system. For example, achieving 100% accuracy awards a special badge. The achievement logic is modular, making it easy to add more challenges later.

My Role

I handled all scripting and systems design for the project, focusing on making the color-matching loop feel responsive and readable.

  • Implemented RGB slider logic (0–255) with synced text fields and color preview.
  • Built the round system with timers, submit logic, and server-side accuracy checks.
  • Created the game summary screen showing server color, player color, and accuracy.
  • Integrated Roblox DataStore to save and load player stats.
  • Added an achievement manager that awards badges like the 100% accuracy reward.
